Установка

Процесс установки данной системы не сложный, но требует внимательности, терпения и бдительности. Данная система состоит из 3-ех программ: Launcher, Update Server (UpdSrv), Launcher Configs(LC).
 
–     Launcher. Эта программа отвечает за обновление игрового клиента и запуск игры. Она работает с файлом: data.p4f (настраивается с помощью LC). А так же ведет постоянное общение с серверной программой UpdSrv для скачивания нужных обновлений и считывания настроек.
 
–     LC. С помощью данной программы Вы можете задать минимальные настройки для Launcher’а(скин, версию, IP-адрес сервера, порт и т.д.), а так же создать свой уникальный дизайн Launcher’а или же локализовать его на нужный Вам язык.
 
–     UpdSrv. Данная программа хранит в себе самые критические настройки, список новых файлов обновления и другие настройки для синхронизированной работы с Launcher’ом.
