Панель управления пакетами

С помощью данной панели Вы можете создавать и управлять пакетами обновлений. Пакеты служат для создания более гибкого обновления. Например, Вы можете создать пакет обновлений "Full" и "Without Sound". В итоге пользователю на выбор будет предоставлено скачивание полного клиента игры или же его версию без звука. Пакеты настраиваются отдельно.

Название пакета

Имя текущего\создаваемого пакета.

Добавление пакета

Добавление нового пакета. Каждый пакет должен иметь уникальное название.

Удаление пакета

Удаление текущего (выбранного) пакета.

Веб-адрес хостинга

Используется только если выбран метод обновления НТТР. Данное поле используется для идентификации директории клиента на веб-хостинге. Например http://mysite.ru/FullClient/ . В итоге ссылки на данный файл будут сформированы относительно этого пути.

Шанс отправки полного клиента

Данное поле устанавливает сколько не хватает % от полного клиента. При построении списка обновлений автоматически создается архив с полным клиентом. Если в данном поле установить 80% то это значит что если пользователю необходимо скачать более 80% игры то автоматически начнется скачивание полного клиента.

Очистка списков

Обновление файлов

Обновить все списки. Данная команда считывает указанную директорию игры и находит в ней все файлы. Генерируются контрольные сумы, размер и при надобности ссылки на файлы. Обновляются списки.
