Редактор дизайна

Клиентская область

Визуальное представление Launcher'a. Все действия которые Вы выполняете сразу отображаются. При загрузке оригинального Launcher'a он будет выглядеть точно также.

Панель редактирования

Пользуясь элементами на настройками панели редактирования Вы сможете изменить дизайн Launcher'a. Панель отвечает за настройку расположения и отображения всех компонентов Launcher'a.

Текущее действие

Выбор действия над объектом. Выбирая одно из трех действий Вы будете по разному действовать на выбранный Вами обьект. Перемещать его, изменять размер или же вывести информацию о нем.

Переключатель Resize

При нажатии на объект Вы будете изменять его текущий размер.

Переключатель Drag & Drop

При нажатии на объект Вы будете изменять его текущее месторасположение.

Переключатель None

При нажатии на объект Вы получите информацию, в правой панели, о текущем объекте.

Текущий шар

Launcher имеет несколько окон. Каждое окно представлено отдельным пунктом. Выбирая пункт Вам будет отображено окно закрепленное за ним. Например пункт "1" отобразит главное окно Launcher'a.

Текущий скин

Список из доступных скинов для преобразования внешнего вида Launcher'a

Название объекта

Текст который отображается на текущем объекте. Вы можете локализовать объект изменяя его название.

Отступ от левого края

Начало расположения объекта за координатой «Х».

Отступ сверху

Начало расположения объекта за координатой «Y».

Высота объекта

Текущая высота объекта

Ширина объекта

Текущая ширина объекта

Дополнительные параметры

Дополнительные параметры существуют для вызова определенных действий объекта после завершения его основных действий. Если Вы хотите чтобы при нажатии на объект открывался нужный веб-адрес достаточно в этом поле вписать, например «web:http://www.rambler.ru» (без кавычек).

Отображение объекта

Вкл.\Выкл. отображение текущего объекта.

Список спрятанных объектов

Список всех спрятанных объектов. Каждый объект можно ввостановить нажав на него а потом на кнопку Visible
