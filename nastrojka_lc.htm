Настройка LC

Основные настройки.
 
     В поля UpdSrv IP и UpdSrv Port надо вписать ІР-адрес ПК на котором установлен UpdSrv(это может быть и доменное имя), и указать порт (порт должен соответствовать порту указанному в UpdSrv). Дальше, по своему вкусу, выбираем Skin(т.н. «шкуры» для изменения внешнего вида Launcher’а), Version(изначальная версия игры, в последствии будет сверятся с версией которая установлена в UpdSrv), и то что будет отображаться в главном окне. Следующим пунктом идет блок Main Panel. Здесь можно установить то что будет отображаться в главном окне Launcher’а. Если Вы хотите чтобы в главном окне отображалась веб-страница то необходимо выбрать пункт Web а в поле URL надо вписать веб-адрес страницы которую, встроенный в Launcher, браузер должен загрузить. Если же выбрать Image то необходимо в поле URL указать путь к рисунку который будет отображаться. Путь можно указывать как прямой (C:\MMOGAME\...) так и не прямой (Images\myimage.jpg) . В последнем варианте будет выбрана текущая папка в которой расположен Launcher и от туда продолжится путь. То есть если у вас Launcher находится в C:\MMOGAME\MyGame\Launcher.exe и вы хотите чтобы рисунок грузился с соседней папки, то вам надо указать просто Images\myimage.jpg , это будет эквивалентно C:\MMOGAME\MyGame\Images\myimage.jpg. Если Вы хотите использовать рисунок с Вашего веб-сервера тогда используйте пункт Web. Так же есть возможность воспроизводить видео файлы. Для этого следует выбрать Movie, а в поле URL вписать путь к видео файлу (как и в случае с картинкой).
 
Изменение интерфейса.
 
     Изначально на второй вкладке  LC отображен стандартный дизайн Launcher’а. Сам же дизайн можно изменять, практически, как угодно. Объекты можно растягивать или перемещать, изменять цветовую гамму или же прятать их. Все манипуляции будут проводится на странице Skin Builder. Подробнее остановимся на каждом из пунктов:
 1)     Изменение размера. Для изменения размера надо в поле Event выбрать Resize. После этого достаточно зажать левой кнопкой мыши на любом объекте на рабочей поверхности и водя мышкой изменить размер объекта к нужному значению. Ширина и высота будут динамически отображаться на правой панели. Когда достигнут нужный размер достаточно отпустить кнопку мыши.
 2)     Перетаскивание объектов. Для изменения позиции объекта в рабочей области надо в поле Event выбрать Drag & Drop. Теперь достаточно нажать на нужный объект и перетащить его в нужное место.  При отпускании кнопки мыши объект будет зафиксирован.
 3)     Информация о объекте. Если же Вам надо только получить текущую информацию о нужном объекте достаточно в поле Event выбрать None. Теперь Вы сможете получить только информацию и при этом не будет риска переместить или изменить нужный Вам объект.
 
     Возле панели Event располагается панель Layer. С помощью данной панели Вы сможете перемещаться между «шарами»  Launcher’а. То есть нажав на 1  Вы увидите главное окно  Launcher’а, нажав на 2 Вы увидите окно настроек и т.д.
     Снизу расположены поля Text, Left, Top, Height, Width, Additional.
 
Text – текст который отображается на текущем объекте.
Left – начало расположения объекта за координатой «Х».
Top – начало расположения объекта за координатой «Y».
Height – высота объекта.
Width – ширина объекта.
Additional – дополнительные свойства объекта.
 
     Данные параметры Вы можете так же изменять вручную. Если Вы хотите чтобы при нажатии на объект открывался нужный веб-адрес достаточно в поле Additional этого объекта вписать, например «web:http://www.rambler.ru» (без кавычек).
 
     Ниже можно увидеть поле Visible и список Invisible Controls. Каждый объект Вы можете спрятать. Для этого достаточно выбрать нужный Вам объект и убрать отметку с поля Visible. Сразу после этого данный объект станет невидим. Каждый невидимый объект будет сразу отображен в списке Invisible Controls. Для того чтобы объект опять стал видим необходимо выбрать его в списке Invisible Controls и поставить отметку в поле Visible. Теперь объект опять станет видимым.
 
Локализация.
 
     Частично локализовать Launcher можно изменяя надписи на объектах на вкладке Skin Builder. Но Launcher использует и другие надписи. Например пользовательские сообщения (уведомления пользователя о какой либо операции) или любой другой текст который не является объектом. Для того чтобы локализовать другие сообщения надо перейти на вкладку Messages. Вы увидите список сообщений. Для изменения нужного сообщения надо выбрать его в списке, оно автоматически отобразится и в нижнем поле Message, и в поле Message изменить его содержимое а затем нажать на кнопку Save.
 
 
Шаблоны.
    
     Шаблоны позволяют Вам сохранять или загружать уже готовые дизайны и локализации для Launcher’а. Шаблоны сохраняются без персональных данных сервера. Если Вы хотите сохранить Ваш дизайн тогда перейдите на вкладку Main Configs и нажмите на кнопку Save Template, после чего сохраните созданный шаблон.  Так же Вы можете использовать уже готовые шаблоны. Для этого зайдите  перейдите на вкладку Main Configs и нажмите на кнопку Load Template. После этого Вам будет предложено открыть файл шаблона. Все файлы шаблонов имеют расширение *.tml.
 
Завершение.
 
     После проделанных работ следует перейти на вкладку Main Configs и нажать на кнопку Save. Сохранить файл в нужном месте. Следует помнить что сохранять надо под именем data.p4f иначе Launcher не увидит файл настроек.
