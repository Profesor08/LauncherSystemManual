Чистка игрового клиента

Вкл\выкл чистку игрового клиента во время обновления последнего. Перед началом обновления (Full Update) Launcher проверит каталог игры на наличие не нужных файлов, и очистит клиент от них.

Тип обновления

Выбор типа обновления игрового клиента
Наверх

HTTP

Обновление через HTTP сервер. Обновление клиента будет происходить скачиванием файлов которые должны быть размещены на НТТР сервере. НТТР сервер может стоять на другом удаленном ПК, главное чтобы файлы которые вы планируете обновлять были и на удаленном ПК. Данный тип обновления несколько не стабилен, медленный и ресурсоемкий.
Наверх

Sockets

Прямое обновление. Launcher закачивает файлы напрямую через UpdSrv. Игровой клиент должен располагаться на том же ПК на котором установлен и UpdSrv. Данный тип обновления работает гораздо быстрее и более стабильный.
Наверх

FTP

Обновление через FTP сервер. Идентично до первого метода. Только вместо протокола НТТР будет использоваться протокол FTP, а вместо веб-сервера будет использоваться FTP сервер.
Наверх

в данном поле необходимо указывать дополнительные параметры для загрузки игры (Game File). Например: /u127.0.0.1 /p44405 (для игры MuOnline).

Файл запуска игры

Указывает относительный путь к файлу для запуска игры. Например если игровой файл находится в том же каталоге что и Launcher, то в данное поле необходимо вписать game.exe(Вам же, соответственно, нужно указать главный файл Вашей игры).
Наверх

Добавление нового фильтра

Данный фильтр добавляет ключевые слова для фильтрации при чистке клиента. Например, если вы хотите чтобы Launcher не удалял скриншоты которые сохранены в папке с игрой необходимо добавить ключевое имя файла в список. Например в поле Name ввести Screenshot и нажать на кнопку Add. После этого будет добавлен новый фильтр. Фильтр работает по принципу поиску файлов в ОС Windows, можно использовать символы «*» , «?» и другие. Внимание! Если вы ввели Screenshot, то фильтр будет распространятся не только на точное имя файла , но и на косвенное! То есть Screenshot1.jpg, Screenshot2222.jpg ScreenshotByVasiaPypkin.exe так же будут подлежать фильтрации (фильтруемые файлы не будут удалены при чистке игрового клиента).
Наверх
