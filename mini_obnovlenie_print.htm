Список контрольных сум

Данный список содержит контрольные сумы файлов. Контрольные сумы высчитываются с помощью алгоритма MD5.

Список размеров файлов 

Список содержит размер файлов в байтах

Список ссылок на файлы

В этом списке находятся все ссылки на файлы для НТТР протокола. Данный список заполняется только если выбрано обновление по HTTP

Веб-адрес хостинга

Используется только если выбран метод обновления НТТР. Данное поле используется для идентификации директории клиента на веб-хостинге. Например http://mysite.ru/FullClient/ . В итоге ссылки на данный файл будут сформированы относительно этого пути.
